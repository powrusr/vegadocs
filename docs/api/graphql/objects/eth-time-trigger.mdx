Trigger for an Ethereum call based on the Ethereum block timestamp. Can be
one-off or repeating.

```graphql
type EthTimeTrigger {
  initial: Timestamp
  every: Int
  until: Timestamp
}
```


### Fields

#### [<code style={{ fontWeight: 'normal' }}>EthTimeTrigger.<b>initial</b></code>](#)<Bullet />[`Timestamp`](/testnet/api/graphql/scalars/timestamp) <Badge class="secondary" text="scalar"/>
> Trigger when the Ethereum time is greater or equal to this time, in Unix seconds.
> 

#### [<code style={{ fontWeight: 'normal' }}>EthTimeTrigger.<b>every</b></code>](#)<Bullet />[`Int`](/testnet/api/graphql/scalars/int) <Badge class="secondary" text="scalar"/>
> Repeat the call every n seconds after the initial call. If no time for
initial call was specified, begin repeating immediately.
> 

#### [<code style={{ fontWeight: 'normal' }}>EthTimeTrigger.<b>until</b></code>](#)<Bullet />[`Timestamp`](/testnet/api/graphql/scalars/timestamp) <Badge class="secondary" text="scalar"/>
> If repeating, stop once Ethereum time is greater than this time, in Unix
seconds. If not set, then repeat indefinitely.
>

List markets in a succession line

```graphql
successorMarkets(
  marketId: ID!
  fullHistory: Boolean
  pagination: Pagination
): SuccessorMarketConnection
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>successorMarkets.<b>marketId</b></code>](#)<Bullet />[`ID!`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> Market ID of any market in the succession line
> 

#### [<code style={{ fontWeight: 'normal' }}>successorMarkets.<b>fullHistory</b></code>](#)<Bullet />[`Boolean`](/testnet/api/graphql/scalars/boolean) <Badge class="secondary" text="scalar"/>
> Flag indicating whether to include the full succession line or not.
If not specified, the default is false.
When false, only the requested market and its children are returned.
> 

#### [<code style={{ fontWeight: 'normal' }}>successorMarkets.<b>pagination</b></code>](#)<Bullet />[`Pagination`](/testnet/api/graphql/inputs/pagination) <Badge class="secondary" text="input"/>
> Optional pagination information
> 

### Type

#### [`SuccessorMarketConnection`](/testnet/api/graphql/objects/successor-market-connection) <Badge class="secondary" text="object"/>
> Connection type for retrieving cursor-based paginated market information

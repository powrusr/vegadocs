Describes the trigger for an auction

```graphql
enum AuctionTrigger {
  AUCTION_TRIGGER_UNSPECIFIED
  AUCTION_TRIGGER_BATCH
  AUCTION_TRIGGER_OPENING
  AUCTION_TRIGGER_PRICE
  AUCTION_TRIGGER_LIQUIDITY_TARGET_NOT_MET
  AUCTION_TRIGGER_UNABLE_TO_DEPLOY_LP_ORDERS
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>AuctionTrigger.<b>AUCTION_TRIGGER_UNSPECIFIED</b></code>](#) 
> Invalid trigger (or no auction)
> 

#### [<code style={{ fontWeight: 'normal' }}>AuctionTrigger.<b>AUCTION_TRIGGER_BATCH</b></code>](#) 
> Auction because market has a frequent batch auction trading mode
> 

#### [<code style={{ fontWeight: 'normal' }}>AuctionTrigger.<b>AUCTION_TRIGGER_OPENING</b></code>](#) 
> Opening auction
> 

#### [<code style={{ fontWeight: 'normal' }}>AuctionTrigger.<b>AUCTION_TRIGGER_PRICE</b></code>](#) 
> Price monitoring
> 

#### [<code style={{ fontWeight: 'normal' }}>AuctionTrigger.<b>AUCTION_TRIGGER_LIQUIDITY_TARGET_NOT_MET</b></code>](#) 
> Liquidity monitoring due to unmet target stake
> 

#### [<code style={{ fontWeight: 'normal' }}>AuctionTrigger.<b>AUCTION_TRIGGER_UNABLE_TO_DEPLOY_LP_ORDERS</b></code>](#) 
> Liquidity monitoring due to not being able to deploy LP orders because there's nothing to peg on one or both sides of the book
>
